Comandos Essenciais

Listar arquivos

ls [opções] [diretório]

Lista os arquivos e diretórios. Opções comuns: -l (lista detalhada), -a (mostra arquivos ocultos).

Mudar de diretório

cd [diretório]

Muda o diretório atual. "cd .." volta um nível, "cd ~" vai para o diretório home.

Criar diretório

mkdir [opções] [nome-do-diretório]

Cria um novo diretório. Opção -p cria diretórios pais se não existirem.

Copiar arquivos

cp [opções] [origem] [destino]

Copia arquivos ou diretórios. Opção -r copia diretórios recursivamente.

Mover/Renomear arquivos

mv [opções] [origem] [destino]

Move ou renomeia arquivos e diretórios.

Remover arquivos

rm [opções] [arquivo]

Remove arquivos. Opção -r remove diretórios e seu conteúdo. CUIDADO: este comando é permanente.

Visualizar processos

ps [opções]

Mostra os processos em execução. Opção aux mostra todos os processos do usuário.

Gerenciador de pacotes (Debian/Ubuntu)

apt [comando] [pacote]

Gerenciador de pacotes para distribuições baseadas em Debian. Comandos comuns: install, remove, update, upgrade.
